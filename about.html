Biografi

Della Rizkyana Okvitaria adalah nama saya. Saya lahir di Banyuwangi, Jawa Timur-Indonesia tepatnya pada tanggal 14 Oktober 2003. Saya tinggal kota Muncar,Banyuwangi bersama keluarga saya. Saya mempunyai saudara perempuan dan saya merupakan anak bungsu di keluarga saya.

Saat ini saya masih kelas X di SMK TELKOM MALANG Angkatan'28. Saya mengambil jurusan Rekayasa Perangkat Lunak karena saya memang tertarik di dunia programming terutama dalam pembuatan aplikasi/software. Itulah sedikit biografi tentang saya. Terimakasih telah bersedia membaca.

Kegemaran

Bermain catur termasuk salah satu hobi saya karena untuk mengasah otak.Dari kecil saya sudah bisa bermain catur karena sering menonton kakak dan ayah bermain catur. Saya belum jago, namun bisa. Sekarang ini saya main caturnya sering online bersama teman saya biasanya.

Jalan-Jalan untuk mengistirahatkan fikiran dan untuk bersenang-senang.Ketika hari libur telah tiba pasti saya sempatkan untuk jalan-jalan meskipun hanya jarak dekat.

Dengan sering membaca maka saya akan membuka jendela dunia. Membaca adalah kegemaran orang-orang sukses yang patut dicontoh. Karena dengan membaca wawasan saya akan luas.Sangat banyak sekali manfaat yang bisa di dapat karena membaca.

Untuk menghilangkan kejenuhan karena tugas yang melanda ataupun gabut saya suka bermain game online.
